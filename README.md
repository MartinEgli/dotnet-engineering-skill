# .NET Engineering Skill

Focused Codex skill for .NET engineering decisions: C#, ASP.NET Core, EF Core,
testing, performance, modernization, Azure integration, Clean Architecture, and
software diagrams.

## Modes

- `/dotnet assess`
- `/dotnet api`
- `/dotnet data`
- `/dotnet clean-architecture`
- `/dotnet testing`
- `/dotnet performance`
- `/dotnet upgrade`
- `/dotnet azure`
- `/dotnet diagram`


## Continuous Improvement

This skill supports an explicit feedback loop. Use /dotnet-engineering feedback to
capture lessons and /dotnet-engineering improve to propose changes. Improvements must
remain traceable, validated, committed on a feature branch, and pushed before
they become part of the released skill behavior.
